## Installation

Run development mode

```js
npm run dev
```

Generate optimized version of app

```js
npm run build
```

Serve locally optimized version

```js
npm run start
```

## Stack

- TypeScript
- React (Next.JS framework)
- Tailwind CSS
- AutoAnimate library
